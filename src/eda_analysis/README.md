# EDA Analysis Notebook

## Overview
Comprehensive exploratory data analysis for the romance novel dataset, providing statistical insights and data quality assessment.

## What This Notebook Does
- **Data Loading & Setup**: Loads and validates the core research dataset (80,755 books, 23 columns)
- **Data Cleaning**: Standardizes data types and handles missing values
- **Statistical Analysis**: Performs normality tests and descriptive statistics
- **Visualization**: Creates distribution plots and outlier analysis charts
- **Data Optimization**: Identifies filtering strategies for statistical reliability
- **Outlier Detection**: Comprehensive analysis of unusual page counts and data quality issues
